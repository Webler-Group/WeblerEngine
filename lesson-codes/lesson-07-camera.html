Lesson 7 — Camera

Two demos: Free Camera (move, rotate, zoom — controls are always screen-aligned) and Camera Follow (smooth player tracking, with optional orientation follow via the up vector).

WASD / arrows — move camera
Q / E — rotate  ·  scroll / +− — zoom

Free Camera

WASD/arrows moves the camera in its own screen-aligned axes — after rotating, W still goes toward screen-up, not world-up.

Q/E (or ↺/↻) change the up vector, rotating the entire view.

// Rotate: keep a running angle camAngle += rotSpeed * dt; // Q/E cam.up = new Vector( Math.sin(camAngle), -Math.cos(camAngle), ); // Move in screen-aligned axes const up = cam.up; const right = new Vector(-up.y, up.x); cam.position.x += right.x * speed * dt; cam.position.y += right.y * speed * dt; cam.position.x += up.x * speed * dt; cam.position.y += up.y * speed * dt;

Camera Follow

The camera smoothly follows the player using exponential decay.

Enable Follow orientation and the up vector rotates so the player always faces screen-up — the world tilts to match the movement direction.

player x0
player y0
cam angle0°
cam.up.x0.00
cam.up.y-1.00
// Position follow (exponential decay) const k = 1 - Math.pow(0.02, dt); cam.position.x += (player.x - cam.position.x) * k; cam.position.y += (player.y - cam.position.y) * k; // Orientation follow — player faces up // target: player._facing + π/2 const diff = wrapAngle( target - camAngle); camAngle += diff * Math.min(1, 5*dt); cam.up = new Vector( Math.sin(camAngle), -Math.cos(camAngle), );
